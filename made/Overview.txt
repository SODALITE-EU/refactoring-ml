- Experiments: Contains the template files for the experiments which are used to generate different unique parameters. Python program can be run to randomly run experiments.

- Infrastructure: Contains all of the services used in the infrastructure. Python program can be used to deploy all of them.

- Load generation: Contains the components necessary for load generation. 

- Models: Consists all of the models trained.

- Data_Scraping_Preparation: Script used to request, prepare, select, and save data.

- Models_XAI: Script used to train models and generate XAI.

- Deployment: Script used to deploy the infrastructure on the K8S cluster.
